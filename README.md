# Tijario Android

Native Android application for Tijario built with Kotlin and Jetpack Compose.

This repository is independent from the existing Tijario web/backend repository. The web/backend project remains the source of truth for product behavior, data model, Supabase configuration, and secure server-side operations.

## Status

Initial repository preparation only. The Android application has not been built or scaffolded yet.

## Intended Stack

- Kotlin
- Jetpack Compose
- Material 3
- Arabic RTL first
- Existing Tijario Supabase project
- Existing secure backend APIs for privileged operations

## Boundaries

Do not copy the web project into this repository. Do not add `.env` files, service-role keys, Replicate tokens, database passwords, signing keys, or backend secrets.
